# 📺 Codyflix - Netflix Clone

This is a high-performance Netflix clone built with **Next.js 13/14**, inspired by the comprehensive tutorial from **Antonio**. The project focuses on modern web development patterns, responsive design, and full-stack integration.

---

## 🚀 Features

- **Dynamic Authentication:** Seamlessly toggle between "Sign In" and "Register" modes using React `useState` and `useCallback`.
- **Reusable Components:** Robust `Input` components built with **TypeScript Interfaces** for type safety and scalability.
- **Modern UI/UX:** Crafted with **Tailwind CSS**, featuring a mobile-responsive layout and Netflix's iconic aesthetic.
- **Client-Side Logic:** Optimized interactivity using the `"use client"` directive in Next.js App Router.

---

## 🛠 Tech Stack

| Technology       | Purpose                          |
| :--------------- | :------------------------------- |
| **Next.js**      | React Framework (App Router)     |
| **TypeScript**   | Static Typing for robust code    |
| **Tailwind CSS** | Utility-first styling            |
| **React Hooks**  | State and Memoization management |
